#README

Diese Endpunkt Doku dient der Evaluation der Projekt in Bezug auf die Güte/Genauigkeit der Empfehlungen.
Sie ist somit keine vollumfassende Doku, sonder jedes Projekt sollten entsprechenden Endpunkt zur Verfügung
stellen, um eine einfache und faire Evaluation zu ermöglichen.

Solltet ihr noch Fehler finden oder Vorschläge für signifikante Verbesserungen haben, gerne an nw@cognify.ai melden.
