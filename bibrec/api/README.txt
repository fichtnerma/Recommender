#README

Dieser Endpunkt Doku dient der Evaluation des Projekts in Bezug auf die Güte/Genauigkeit der Empfehlungen.
Sie ist somit keine voll umfassende Doku, sondern jedes Projekt sollten entsprechenden Endpunkt zur Verfügung
stellen, um eine einfache und faire Evaluation zu ermöglichen.

Solltet ihr noch Fehler finden oder Vorschläge für signifikante Verbesserungen haben, gerne an nw@cognify.ai melden.

## Updates

* Jedes Team stellt bis zum Abgabetermin eine Liste der unterstützen Items zur Verfügung (als CSV oder DataFrame in eine PKL-File)
* ItemID als String um den in den Daten verwendeten ISBN-10 Standard zu unterstützen
* wie definiert werden Parameter via Query mitgegeben
